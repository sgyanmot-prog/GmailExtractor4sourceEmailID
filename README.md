# GmailExtractor4sourceEmailID

How to Download:
1. Click on the green Code button
2. Click download zip file
3. Click on gmail_extractor_app.html in your files
   
How Your Team Can Use It:
1. Enter their Gmail address (e.g., john.doe@company.com)
2. Select their preferred scan scope (inbox, entire mailbox, or time-based)
3. Click "Start Extraction" — Agent authenticates with their Gmail account
4. Get results — View stats on threads scanned and unique senders found
5. Create Google Sheet — One-click export to Google Drive
Or download as CSV — Save locally for further analysis
